# Task-to-script guide

Use these helpers when you want the repo to generate a useful starting point quickly.

1. Run tools/scaffolding/Generate-NextScript.ps1 -Task "blocking" -Category "performance-troubleshooting" to create a starter SQL script.
2. Run tools/scaffolding/Generate-NextPowerShell.ps1 -Task "backup age" -Category "backups-and-recovery" to create a starter PowerShell helper.
3. Use tools/triage/Quick-TaskRouter.ps1 to find the best category first.
4. Then refine the script with your real DBA logic.
