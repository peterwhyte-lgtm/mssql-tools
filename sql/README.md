# sql

SQL Server scripts grouped for diagnostics, maintenance, and production DBA operations.

## Subfolders

- backups-and-recovery/ - backup coverage, restore, and backup-history checks
- configuration-and-environment/ - instance, memory, MAXDOP, jobs, and services reviews
- high-availability-and-disaster-recovery/ - AG and replica health checks
- maintenance-and-reliability/ - growth, health, and reliability diagnostics
- performance-troubleshooting/ - waits, blocking, CPU, fragmentation, and query analysis
- security-and-permissions/ - logins, roles, and permissions audit helpers
- storage-capacity-management/ - file, log, disk, and capacity checks

The catalog in docs/catalog.md has been refreshed to reflect the current GitHub-style script set.
